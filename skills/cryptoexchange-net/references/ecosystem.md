## Choosing Packages

- Use one exchange package for one exchange.
- Use multiple exchange packages when the app needs only a small fixed set.
- Use `CryptoClients.Net` when the app needs broad exchange coverage from one bundle.
- Do not install `CryptoExchange.Net` alone for exchange access; it is the base library.
